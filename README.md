# ⛅ Weather App

A sleek and interactive weather forecasting tool that provides real-time data for cities worldwide.

## 🛠️ Tech Stack
- **Language:** JavaScript (ES6+)
- **API:** OpenWeatherMap API
- **Styling:** CSS3 (Flexbox/Grid for responsiveness)
- **Structure:** Semantic HTML5

## 🚀 Key Features
- **Instant Search:** Get weather data by entering any city name.
- **Visual Feedback:** Dynamic icons that change based on weather conditions (Sunny, Cloudy, Rain, etc.).
- **Detailed Metrics:** Displays temperature, humidity, pressure, and wind speed.
- **Responsive Design:** Optimized for mobile, tablet, and desktop views.

## ⚙️ Development Highlights
- Handled JSON data parsing from RESTful API responses.
- Managed API keys securely within the development workflow.
- Implemented conditional rendering to change background themes based on the weather status.
